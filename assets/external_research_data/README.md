# External Research Datasets

This directory contains external datasets downloaded for drug interaction analysis and development.

## Datasets

### 1. WHO ATC/DDD (2024-07-31)
- **File**: `WHO_ATC_DDD_2024.csv`
- **Source**: Scraped from the WHO Collaborating Centre for Drug Statistics Methodology.
- **Provider**: [fabkury/atcd (GitHub)](https://github.com/fabkury/atcd)
- **Description**: Anatomical Therapeutic Chemical classification system with Defined Daily Doses.

### 2. DDInter 2.0 (2024-05-14)
- **Files**: `ddinter_downloads_code_*.csv` (Categorized by ATC groups A, B, D, H, L, P, R, V)
- **Source**: [DDInter 2.0 Official Website](https://ddinter2.scbdd.com)
- **Description**: Comprehensive drug-drug interaction database with 300k+ records.

### 3. FooDrugs v4 (2023-07-28)
- **File**: `FinalFooDrugs_v4.sql`
- **Source**: [IMDEA Food Institute / FooDrugs](https://imdeafoodcompubio.com/FooDrugs/)
- **Description**: Food-drug interactions database containing over 500k interactions extracted from textual documents and gene expression data.
