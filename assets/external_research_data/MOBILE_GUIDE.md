# دليل تشغيل السكربت عبر الموبايل (نظام أندرويد)

استخدام الموبايل هو أسرع وأذكى وسيلة لتغيير عنوان الـ IP باستمرار وتجنب الحظر، لأن شركات الاتصالات تمنحك IP جديداً في كل مرة تقوم فيها بتنشيط "نمط الطيران".

## الخطوة الأولى: التجهيز
1. قم بتحميل تطبيق **Termux** من متجر F-Droid (أو Google Play).
2. افتح التطبيق واكتب الأوامر التالية بالترتيب لتثبيت لغة Python والمكتبات المطلوبة:
```bash
pkg update && pkg upgrade
pkg install python
pip install requests beautifulsoup4
```

## الخطوة الثانية: نقل الملفات (طريقة النسخ واللصق)

هناك طريقتان لوضع السكربت داخل Termux، اختر الأسهل لك:

### الطريقة الأولى: النسخ واللصق اليدوي (باستخدام محرر Nano)
1. قم بنسخ كود السكربت بالكامل من متصفح الموبايل (أو من الويندوز وأرسله لنفسك واتساب).
2. في Termux، اكتب الأمر التالي لفتح محرر نصوص جديد:
```bash
pkg install nano
nano bulk_scraper_v7_turbo.py
```
3. **للصق الكود:** اضغط ضغطة مطولة (Long Press) في أي مكان في شاشة Termux السوداء، ستظهر لك قائمة، اختر منها **Paste**.
4. **للحفظ:** اضغط على زر `Volume Down + O` في لوحة المفاتيح (أو اضغط `Ctrl + O` إذا كنت تستخدم لوحة مفاتيح خارجية) ثم اضغط **Enter**.
5. **للخروج:** اضغط `Volume Down + X` (أو `Ctrl + X`).

*كرر نفس الخطوات لإنشاء ملف `unique_drugs.json` ووضع محتواه بداخله.*

### الطريقة الثانية: الوصول لملفات الموبايل مباشرة (أسهل إذا حملت الملفات)
إذا قمت بتحميل الملفات في مجلد الـ Downloads في موبايلك:
1. اعطِ Termux صلاحية الوصول للذاكرة:
```bash
termux-setup-storage
```
*(ستظهر لك رسالة تأكيد على شاشة الموبايل، اضغط Allow).*
2. الآن يمكنك نسخ الملفات من المجلد العادي إلى Termux:
```bash
cp /sdcard/Download/py/bulk_scraper_v7_turbo.py ~/
cp /sdcard/Download/py/unique_drugs.json ~/
```

## أين يتم حفظ النتائج وكيف أصل إليها؟

السكربت يقوم بحفظ النتائج في نفس المجلد الذي يعمل فيه داخل Termux. الملفات الناتجة هي:
1. `ddinter_exhaustive_v7_turbo.json` (البيانات الكاملة)
2. `ddinter_drugs_metadata_v7_turbo.csv` (بيانات الأدوية)
3. `ddinter_interactions_v7_turbo.csv` (بيانات التفاعلات)

### كيف تنقل الملفات لذاكرة الموبايل العادية؟
لرؤية الملفات في تطبيق "ملفاتي" (My Files) أو نسخها للكمبيوتر، استخدم هذه الأوامر لنقلها لمجلد التحميلات (Download):
```bash
cp ddinter_*_v7_turbo.* /sdcard/Download/
```
بعد تشغيل هذا الأمر، ستجد الملفات موجودة في مجلد الـ **Download** داخل موبايلك ومستعدة للنقل لأي مكان آخر.
```bash
python bulk_scraper_v7_turbo.py
```
2. راقب السجلات، إذا لاحظت أن السكربت بدأ يعطي أخطاء (Errors) أو توقف عن جلب البيانات:
    - قم بإيقاف السكربت (اضغط `Ctrl + C`).
    - **فعل "نمط الطيران" (Airplane Mode) لمدة 5 ثوانٍ ثم أغلقه.**
    - انتظر حتى تعود خدمة البيانات (Mobile Data).
    - أعد تشغيل السكربت؛ سيتعرف تلقائياً على آخر دواء توقف عنده ويكمل العمل بـ IP جديد!

## لماذا هذه الطريقة هي الأفضل؟
- **IP Dynamic:** شركات الموبايل لديها آلاف العناوين، وبمجرد قفل الشبكة وفتحها يتغير هويتك تماماً بالنسبة للموقع.
- **السهولة:** يمكنك تشغيل السكربت وترك الموبايل بجانبك والقيام بعملية تغيير الـ IP يدوياً كل نصف ساعة مثلاً لضمان أعلى سرعة.
